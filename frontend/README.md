# PipelineX Frontend

Minimal Next.js + TypeScript frontend for PipelineX with Tailwind.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Project Structure

```
src/app/layout.tsx → Root layout

src/app/page.tsx → Minimal homepage

src/styles/globals.css → Tailwind CSS imports
```
